# ImersaoReactAlura
Projeto criado durante a imersão do alura


### Tecnologias usadas
1. Supabase
2. nextJs
3. Skynet


### Instruções para execução

```
yarn  ( instalação do controlador de pacotes )
yarn dev  ( subir o servidor localmente para uso )
```
